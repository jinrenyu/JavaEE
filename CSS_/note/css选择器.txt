1. 什么是CSS选择器

它是一种CSS中的选择元素的机制,作用是标示出CSS属性所要影响的元素标签的范围.
它可以分为以下5类:
1. 元素选择器
2. 关系选择器
3. 属性选择器
4. 伪类选择器
5. 伪元素选择器
选择器之间可以组合使用, 并列关系使用逗号(,)进行分割.

2. 元素选择器
a.) 通配符选择: 语法   * {}
作用: 选择页面中所有的元素
b.) 标签选择器: 语法   标签名 {}
作用: 选择指定的标签, 会出现分组的写法
c.) 类选择器: 语法     E.类名 {}
作用: E代表标签或者通配符, 选择元素中class属性为类名的标签
d.) id选择器: 语法     E#id名称{}
作用: E代表标签或者通配符, 选择元素中id属性为id名的标签

3. 关系选择符
a.) E>F (表示影响E选择结果的F子结果)
b.) E F(表示影响E所选择结果下面的全部F子结果)
c.) E~F(表示影响E元素下面的兄弟元素F)
d.) E+F(表示影响E元素相邻的兄弟元素F)

4. 属性选择器
a.) E[attr](选择E标签且具有attr属性的元素)
b.) E[attr=''](选择E标签且attr等于某个值的元素)

= 表示相等
^= 表示以某个字符串开头
$= 表示以某个字符串结尾
*= 表示包含某个字符串
~= 表示其中的某一个单词为某串字符串
|= 表示以某一个字符串+'-'的方式开头的

5. 伪类选择器
a:link 表示一个超链接未被访问
a:visited 表示一个超链接被点击过之后
a:hover 当鼠标划过超链接时的样子
a:active 当鼠标点击并且未进行释放时的样子

注意: 为了保证这些伪类能够正确的被执行, 它们的出现顺序必须是link->visited->hover->active
l(link)ov(visited)e h(hover)a(active)te 喜欢 讨厌

6. 伪对象选择器
:before
:after
这两个伪类选择符必须搭配content这个CSS属性进行运用

表示在每个p标签之前都插入一副图片,test.jpg
p:before {
    content: url("../img/test.jpg");
}

>>>--选择器的优先级以及其对应的数值, 数值越大优先级越高
继承的元素样式 --> 没有特殊性, 最低
*、结合符(+/>) --> 最低, 对数值没有贡献
标签、伪元素 --> 1
class、属性选择符、伪类 --> 10
id --> 100
内联式样式 --> 1000
!important --> 无穷大

数值相同时, 出现的顺序越晚其优先级越高
